# Christina Borreby Website

Hesteassisteret lederudvikling i Nordsjælland.

## Setup

1. Pak zip-filen ud
2. Åbn mappen i terminal
3. Kør:

```bash
npm install
npm run dev
```

4. Åbn http://localhost:3000 i browseren

## Deployment

For at bygge til produktion:

```bash
npm run build
```

Output findes i `out/` mappen og kan uploades til enhver statisk hosting (Netlify, Vercel, etc.)
